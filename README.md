## 📂 Project Structure

Every project in this repo follows the same folder structure:

```
Project Name/
├── notebook.ipynb                 # Data cleaning, EDA & analysis (Python)
├── dataset.csv                    # Raw or processed dataset
├── dashboard.pbix                 # Power BI dashboard
├── project_report.pdf             # Written report with findings & recommendations
├── presentation.pptx              # Stakeholder presentation
└── README.md                      # Project-specific documentation
```

---

## 🧠 What Each Project Covers

Each project is built around a real business problem and walks through:

- **Problem Definition** — A clear business question the analysis is meant to answer
- **Data Preparation** — Cleaning, handling nulls, renaming columns, removing redundancies
- **Exploratory Data Analysis** — Understanding distributions, relationships, and outliers
- **Feature Engineering** — Creating new columns that make analysis more meaningful
- **SQL Analysis** — Structured querying for business insights
- **Visualization** — Interactive dashboards in Power BI
- **Report & Presentation** — Written findings and stakeholder-ready slides
- **GitHub** — All code, data, and files version-controlled and documented
